# 题记
 每周新增一篇文章，认真对待。勤而作，共勉之.
 
 对每一个人的文章进行认真细致的查阅,品读, 是对作者最好的尊重!!!
 
 本项目是日常逛掘金社区然后挑选高品质文章,然后收藏于spring列表中, 最后每周从列表中抽取十篇文章进行细读并吸取有效内容后存储于自己的项目中.
